Rocket Man

Rocket Man(2017) is the project designed to convey a message and convince a specific person. The character in the game is Kim Jong-un, North Korean President, and this project aims to deliver a message that if missiles is developed and fired continuously, the damage will eventually return to the North Korea. The game reflects the situation of the times when relations between South and North Korea rapidly deteriorated in 2017.
